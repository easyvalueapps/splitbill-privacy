🔒 Politique de Confidentialité - SplitBill

📅 Dernière mise à jour : 6 août 2025
📱 Application : SplitBill - Easy Bill Splitting
📦 Package : com.easyvalueapps.splitbill
🏢 Développeur : EasyValue Apps

🎯 Résumé Exécutif

SplitBill est une application de partage de factures qui priorise votre confidentialité. Toutes les données sont stockées localement sur votre appareil et aucune information personnelle n'est envoyée vers des serveurs externes.

📊 Données que nous Collectons

SplitBill ne collecte que les données strictement nécessaires au fonctionnement :

  • Calculs de partage : Montants des factures, participants, pourboires (stockés localement)
  • Historique des calculs : Pour faciliter la réutilisation (stocké localement)
  • Préférences utilisateur : Thème (clair/sombre), langue, devise par défaut
  • Paramètres de l'app : Pour maintenir vos préférences entre les sessions
⚠️ Important : Toutes ces données restent exclusivement sur votre appareil. Nous n'avons pas accès à ces informations et ne les collectons pas.

❌ Données que nous ne Collectons PAS

SplitBill ne collecte PAS, ne stocke PAS et n'a PAS accès aux :

  • ❌ Informations personnelles sensibles (nom, numéro de sécurité sociale, adresse)
  • ❌ Données de localisation (GPS, localisation approximative)
  • ❌ Informations financières réelles (cartes de crédit, comptes bancaires, mots de passe)
  • ❌ Contacts téléphoniques (carnet d'adresses, numéros, emails)
  • ❌ Photos ou caméra (fonctionnalité désactivée dans la version 1.0)
  • ❌ Historique de navigation ou données d'autres applications
  • ❌ Identifiants uniques de l'appareil
  • ❌ Données d'utilisation ou analyses comportementales

💾 Stockage des Données

🔐 Politique de Stockage Local :
  • Toutes les données sont stockées localement sur votre appareil
  • Aucune information n'est envoyée vers des serveurs externes
  • Nous ne maintenons pas de sauvegardes cloud de vos données
  • Les données sont stockées dans le répertoire protégé de l'application
  • Les autres applications ne peuvent pas accéder aux données de SplitBill

🤝 Partage des Données

POLITIQUE ZÉRO PARTAGE :

  • ✅ Nous ne partageons PAS de données avec des tiers
  • ✅ Nous ne vendons PAS d'informations utilisateur
  • ✅ Nous n'avons PAS de partenariats de partage de données
  • ✅ Nous n'utilisons PAS de réseaux publicitaires collectant des données
  • ✅ Nous n'intégrons PAS de plateformes d'analyse invasives
📤 Partage Volontaire : Le seul "partage" se produit lorsque vous choisissez d'envoyer des résultats via WhatsApp, email, etc. Dans ce cas, seul le texte formaté du calcul est partagé via le système de partage natif d'Android.

📱 Autorisations de l'Application

SplitBill utilise uniquement les autorisations minimales nécessaires :

✅ Autorisations Actives :

  • WRITE_EXTERNAL_STORAGE : Pour sauvegarder l'historique des calculs localement
  • READ_EXTERNAL_STORAGE : Pour accéder à l'historique sauvegardé
  • INTERNET : Pour les futures mises à jour (v1.1) - actuellement non utilisée

❌ Autorisations NON Utilisées :

  • ❌ Caméra (désactivée dans v1.0)
  • ❌ Microphone
  • ❌ Localisation (GPS)
  • ❌ Contacts
  • ❌ SMS/Téléphone
  • ❌ Calendrier

🗑️ Suppression des Données

Comment supprimer vos données :

  • Méthode 1 : Désinstaller l'application supprime automatiquement toutes les données
  • Méthode 2 : Effacer les données dans les paramètres Android
  • Méthode 3 : Utiliser la fonction "Effacer l'Historique" dans l'app (quand disponible)
🔄 Suppression Garantie : Puisque toutes les données restent sur votre appareil, la désinstallation supprime 100% des informations liées à SplitBill.

👶 Protection des Mineurs

SplitBill convient à tous les âges :

  • ✅ Ne collecte PAS de données de mineurs
  • ✅ Ne contient PAS de contenu inapproprié
  • ✅ Classé tout public (PEGI : 3+)
  • ✅ Outil éducatif pour apprendre la division et les mathématiques

🌍 Transferts Internationaux

NON APPLICABLE : Puisque toutes les données restent localement sur votre appareil, il n'y a pas de transferts internationaux de données.
